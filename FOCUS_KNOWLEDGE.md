# Focus Web - Project Knowledge Base

Tento soubor obsahuje kompletní dokumentaci, filozofii a technické detaily k webu **Focus**, které jsme společně vytvořili během naší vývojové session. Slouží jako referenční bod pro budoucí vývoj.

## 1. Filozofie a Vize
Cílem webu není jen prodat aplikaci, ale **vyvolat emoci**. Žijeme v přehlceném světě plném multitaskingu a notifikací. Aplikace Focus slouží k nemilosrdnému blokování rozptýlení a podpoře hluboké práce. Web tuto filozofii zrcadlí skrze svůj design:
- **Tmavý, prémiový vzhled**: Zastupuje klid a "prázdnotu" prostoru bez rozptýlení.
- **Tekoucí animace a plynulost**: Symbolizuje čistou, ničím nerušenou mysl (plynoucí vlnu světla).
- **Glassmorphism**: Decentní poloprůhledné prvky dodávají moderní hloubku bez zbytečného šumu.

## 2. Technologický Stack
- **Framework**: React 18 + TypeScript + Vite
- **Stylování**: Tailwind CSS v4
- **Animace**: Framer Motion (všechny fade-ins, scroll odhalení a komponenty).
- **WebGL**: OGL (použito pro `SoftAurora.tsx` shader).
- **Routování**: `react-router-dom` (SPA navigace).
- **Lokalizace**: `i18next` + `react-i18next` (Čeština a Angličtina).
- **Ikony**: `lucide-react`.

## 3. Architektura a Struktura Cest
Web je postaven jako Single Page Application (SPA).
- `/` - **Home Page**: Masivní "Earn your focus." nadpis, vysvětlení features, `AppDemo` (ukázka chatu/AI), `PhilosophyScroll` (manifesto).
- `/about` - **About Page**: Rozšířené informace o aplikaci, plné manifesto, pricing nebo vize (připraveno na budoucí rozšíření).
- `/download` - **Download Page**: Jednoduchá a čistá stránka s instrukcemi ke stažení (Windows/Mac/Linux).

Vše obaluje `Layout.tsx`, který drží **globální navigaci**, **ambientní bubliny v pozadí** a **footer**, aby se nemusely tyto těžké prvky překreslovat při přechodu mezi stránkami.

## 4. Klíčové Vizuální Prvky
1. **Ambientní Blobs (`index.css` & `Layout.tsx`)**: 
   Dvě obří, rotující kuličky s masivním `blur(100px)` filtrem. Vytváří neustálý, pomalu se měnící fialovo-růžový opar v pozadí.
2. **Soft Aurora (`SoftAurora.tsx` & `AuroraIntro.tsx`)**: 
   Matematický WebGL shader kreslící "auroru" (světelnou vlnu). Na hlavní stránce je tato vlna zpočátku skrytá a po načtení se pomalu "vykreslí" (odkryje zleva doprava pomocí Feathered CSS masky). Zpracováno s fallbackem na `mediump` pro starší mobily.
3. **AppDemo (`AppDemo.tsx`)**: 
   Falešné, interaktivní okno aplikace, které simuluje funkčnost Focusu. Zobrazuje "AI Evaluaci" – psací efekt (typewriter), kde se AI ptá uživatele a uděluje mu XP.
4. **Philosophy Scroll (`PhilosophyScroll.tsx`)**: 
   Epická sekce s manifestem. Jak uživatel scroluje, texty mají parallax efekt a na pozadí se pomalu kreslí "tepající" neonová linka symbolizující cestu soustředění.

## 5. Průběh a Optimalizace
Při vývoji jsme museli vyřešit několik složitých technických a vizuálních problémů:
- **First Load Animace**: Úvodní epické animace (kreslení Aurory a vynoření nadpisu) se nesmí přehrávat pokaždé, když uživatel klikne na `O aplikaci` a zpět. Vyřešili jsme to přes lokální stav `initialLoad` (uložený v paměti DOMu pomocí `useRef` a zpožděný).
- **Overscroll na iOS**: Při tahání obrazovky "pod footer" iPhony ukazovaly useknutý gradient. Opraveno nastavením `bg-focus-bg` výhradně pro `html` (tím se obarví "dno" Safari) a odebráním barvy z `body`, aby se neskryly prvky s negativním z-indexem.
- **Mobilní Responzivita**: Blobs (bubliny) se na mobilu zmenšovaly tak moc, že je blur efekt vymazal. Ošetřeno pomocí `min-width: 500px`. Stejně tak gigantické fonty (např. 9rem) mají fallbacky (`text-5xl`) na menších displejích.
- **Hamburger Menu**: Přidáno plnohodnotné mobilní "glassmorphic" overlay menu v `Layout.tsx`, ovládané přes stav `isMenuOpen`.
- **SPA na Vercelu**: Aby fungoval refresh na `/about`, přidali jsme `vercel.json` s `"rewrites"`, který všechny dotazy přesměruje do `index.html`.
